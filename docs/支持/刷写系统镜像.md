# 刷写系统镜像

> 基本原则：将系统备份好的镜像文件刷写到工控机
> 声明：当工控机系统被破坏、缺失文件导致无法运行自动驾驶或在黑屏导致无法进入系统时，可以用U盘来还原工控机系统恢复出场设置。
> 参考资料：<https://www.youtube.com/watch?v=YEFt2LPSYMk>
