# 备份系统镜像
 
> 基本原则：将工控机系统制作镜像  
> 声明：避免用户在工控机里部署其他的驱动或在软件时破坏系统配置导致无法运行自动驾驶及启动驱动，可以通过备份的镜像来还原最初系统（系统的最初镜像已附带在U盘里）。用户也可以备份自己的系统镜像，可以用硬盘来保存工控镜像。  
> 参考资料：<https://www.youtube.com/watch?v=8-7w5zwD9M4>
